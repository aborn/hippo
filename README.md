# Hippo 河马--运营配置平台

## 包说明
hippo-admin 提供给后台管理端使用的M端，war类型  
hippo-api 提供给SDK使用的api应用，接口流量大，war类型  
hippo-contract 定义的Dubbo接口，jar类型  
hippo-node 前端页面代码模块，TODO @lxl1024  
hippo-repository 数据操作包，获取数据库的接口等  
hippo-sdk 提供给业务使用的sdk，jar类型
